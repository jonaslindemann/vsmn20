# Links

## Books and tutorials

* [How to Think Like a Computer Scientist](https://runestone.academy/ns/books/published/thinkcspy/index.html) — Recommended reading before Worksheet 1. Covers Python basics up to recursion.
* [Python tutorial](https://docs.python.org/3/tutorial/) — Official Python language tutorial and reference.
* [NumPy tutorial](https://numpy.org/learn/) — Covers array creation, indexing, and linear algebra used in Worksheets 1–5.
* [Matplotlib user's guide](https://matplotlib.org/stable/users/index) — Plotting library used for graphs and visualisation in Worksheets 1–3.
* [Ingenjörens guide till Python](http://guidetillpython.se/) — Swedish introduction to Python for engineering students.
* [CALFEM for Python Manual](https://calfem-python-manual.readthedocs.io/en/latest/) — Primary reference for all FEM routines used in Worksheets 2–5.

## Software

* [Conda-Forge Python distribution](https://conda-forge.org/) — Recommended Python installation. Includes NumPy, Matplotlib, and conda package management.
* [Google Colab - Notebooks](https://colab.research.google.com/) — Browser-based Python environment, useful for Worksheet 1 without a local installation.
* [Visual Studio Code](https://code.visualstudio.com/Download) — Recommended code editor for writing and debugging Python files.
* [ParaView](https://www.paraview.org/download/) — 3D visualisation software required for Worksheet 5 VTK result files.
